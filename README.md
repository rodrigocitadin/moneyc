# Moneyc

Just for knowledge purposes with React, Typescript, Tailwind and some good libraries

## How to run?

First, clone:

`git clone https://github.com/rodrigocitadin/moneyc && cd moneyc`

Then, install deps:

`npm i`

So, run the server:

`npm run dev:server`

And run the front

`npm run dev`

Go to `localhost:5173` in your browser and done!
